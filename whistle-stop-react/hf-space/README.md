# The Whistle Stop 💈

A modern companion for classic craftsmanship. Book appointments, shop products, manage your barbershop — all from one platform.

**Features:**
- 📅 Book appointments with mandatory payment processing
- 🪑 Real-time chair availability queue with SMS notifications
- 🛒 Shopping cart with secure checkout
- 🔔 Push-style notifications for chair readiness
- 📱 Fully responsive mobile UI
- 🎬 Animated background slideshow
